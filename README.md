# 🧱 빨간벽돌 집짓기 경매

하나의 단일 HTML 파일로 구성된 오프라인 형태의 '정적 호스팅' 기반 경매 게임 웹사이트입니다. 교사가 학급별로 여러 팀을 구성하고 각 재료를 경매(혹은 배정) 방식으로 분배하여 최종 비용을 계산하는 교육활동용 프로그램입니다.

## 🌟 주요 기능 설명 및 업데이트 결과

*   **다중 슬롯 저장소 (Multiple Backup Slots)**
    *   별도의 백엔드(DB) 없이 개별 수업 컴퓨터의 브라우저 저장소(localStorage)를 활용하여 **학급별(1반, 2반 등) 진행 상황을 독립적으로 저장**합니다.
    *   창이 무단으로 종료되더라도 시작 시 '이전 수업 불러오기'를 통해 손실 없이 복구할 수 있습니다.

## 🚀 사용법


1.  **⚙️ 수업 설정 탭**에서 진행할 학급의 이름(저장 슬롯)을 입력하고 팀 수를 셋팅한 뒤 수업을 엽니다.
2.  **🔨 경매 진행 탭**에서 아이템별로 학생들의 입찰액을 입력받아 재료를 판매합니다.
3.  **🏆 최종 결과 탭**에서 팀별 구매 결과를 요약 및 우승팀을 산정합니다.
